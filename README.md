# JHContacts
快捷保存手机号码到系统通讯录中的需求在很多的应用中都会用的到，QQ、微信等社交软件都是可以见到的，虽然实现起来也是很简单的，小编还是把这个小功能整理一下，方便后面在需要的时候能方便的使用，也能方便朋友们能感到方便。有需要的直接可以拿去，甚是方便，废话不多说，代码已经上传Github：[JHContacts](https://github.com/Gjianhao/JHContacts)
